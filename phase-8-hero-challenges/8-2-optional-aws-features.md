# 8-2 Optional AWS Features

## Overview

Integrate optional AWS features like IAM policies, CloudWatch, Lambda triggers.

## Code Example

* Add optional IAM policy module
* Add CloudWatch alarms for EC2 or S3 events
* Add Lambda for automation

Commands:

```bash
terraform init
terraform plan
terraform apply
```

## Expected Output

* Optional AWS features deployed successfully
* Resources configured per best practices

## Insights

* **Why this demo exists:** Showcases optional enhancements for infrastructure.
* **Key points:** Keep optional features modular and parameterized.
* **Common mistakes / pitfalls:** Overcomplicating core architecture.
